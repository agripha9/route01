API 키 설정
Anthropic API 키를 입력하면 바로 테스트할 수 있어요. 키는 이 브라우저에만 저장되며 외부로 전송되지 않습니다.
키를 입력하고 저장하세요.
키 발급 방법
  1. console.anthropic.com 접속 → 로그인
  2. 왼쪽 메뉴 API Keys → Create Key 클릭
  3. 키 복사 후 위 입력란에 붙여넣기
멘토 스타일 선택
원하는 피드백 톤을 선택하면, 이후 답변에 바로 적용됩니다.
